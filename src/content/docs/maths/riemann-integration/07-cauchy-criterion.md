## Theorem

A bounded function $f:[a, b] \rightarrow R$ is Riemann integrable **iff** for
every $\epsilon > 0$ there exists a partition $P_\epsilon$ of $[a,b]$, which may
depend on $\epsilon$, such that:

```math
U(f, Pϵ) − L(f, Pϵ) ≤ \epsilon
```
