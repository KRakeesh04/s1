Defined as below for $n\gt 0$: $ $

```math
\Gamma(n)=\int_0^\infty e^{-x}x^{n-1}\,\text{d}x
```

Aka. Eulerian integral of the second kind.

:::note

For $n\le 0$, the gamma function is divergent.

:::

## Properties

- $\Gamma(1)=1$
- $\Gamma(n+1)=n\Gamma(n)$
- $\Gamma(n+1)=n!$
